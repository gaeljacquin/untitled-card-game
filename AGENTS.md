# AGENTS.md

Untitled Card Game (UCG) is a single-player card puzzle game where players arrange cards on a grid to score poker hands.

## Essentials
- Package manager: `bun` (see `bun.lock`)
- Primary commands: `bun dev`, `bun build`, `bun test`, `bun lint`, `bun type-check`
