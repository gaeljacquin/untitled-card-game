# Web App

- Framework: React 19 + Vite + React Router
- UI: shadcn/ui, Tailwind CSS, Magic UI
- State: Zustand with persist and devtools middleware
- Real-time: Socket.IO client
- Drag-and-drop: @dnd-kit
- Views in `apps/web/src/views`: home, ab-mode, how-to-play, settings, credits, not-found
- Routes: `/`, `/game/four`, `/game/five`, `/how-to-play`, `/settings`, `/credits`
